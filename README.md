### 1-2.nvmの動作確認のため、コマンドプロンプトを起動し、以下のコマンドを実行→バージョンが表示されることを確認

>```
>nvm -v
>```

### 1-3.以下のコマンドで入手可能なNodeを調べる。

>nvm list available
>```
